# Skills Index

Project-local skills can be stored in this folder.

## Recommended Structure

- `_templates/SKILL.template.md` as the baseline
- one folder per skill with `SKILL.md`

## Good Candidates For Personality

- Telegram production smoke
- reflection-task triage
- memory or profile data inspection
- Coolify deploy verification
- `adopt_template_into_existing_project/` guides safe migration of an existing repository onto this agent workflow.
